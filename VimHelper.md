### Замена текста
```
:s/<regexp>/<replacement>/ # и нажмите <ENTER>, где <regexp> - что хотите заменить, и <replacement> - на что.
:%s/<regexp>/<replacement>/g # замена каждого вхождения образца во всем файле (что обычно и требуется);
:%s/<regexp>/<replacement>/gc # подтверждение потребность замены в каждом конкретном случае при выполнении глобального замещения (здесь «c» означает «confirm»).
```
### Отступ
```
:set autoindent – включить авто отступ;
:set tabstop=(число) – установить авто табуляцию (где число есть число символов табуляции).
# Если авто отступ включен, вы можете использовать <C-d> для уменьшения и <C-t> для увеличения уровня отступа. 
```
### Поиск символа в строке 
```
f<символ> # вправо
F<символ> # влево
t<символ> # курсор перед символом
; # повторяет поиск в прямом направлении
, # повторяет поиск в обратном направлении
```
### Поиск слова в тексте
```

?<слово> # в обратном направлении
n # продолжает поиск 
N # в обратном направлении
```

### Перемещение по словам
```
W — первый символ следующего большого слова;
E — следующий последний символ большого слова;
B — предшествующий первый символ большого слова.

w – перемещения на первый символ следующего слова нажмите;
e – передвинуться на следующий последний символ слова нажмите;
b – сдвинуться на предыдущий первый символ слова можно нажимая.
```

### Файловые команды редактора Vim
#### Сохранить редактируемый файл
`:w`
#### Записать файл под новым именем
`:w <name>`
#### Сохранить, игнорируя флаг read only
`:w!`
#### Сохранить и выйти 
```
:wq
:x
```
#### Выйти
`:q`
#### Выйти без сохранения
`:q!`
#### Прочитать содержимое файла и вставить его в редактируемый текст 
`:r <name>`
#### Начать редактирование еще одного файла 
`:e <name>`
#### Список редактируемых файлов (активных буферов)
`:ls`
#### Перейти к буферу номер N
`:b <N>`
